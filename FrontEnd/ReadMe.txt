**
Ouvrire le Terminal
**
Installez le serveur HTTP en tapant la commande suivante :
npm install -g http-server
**
Une fois l'installation terminée, lancez le serveur avec la commande suivante :
http-server
**
Important : Ne fermez pas le terminal après avoir lancé le serveur.
**
Ouvrez ensuite votre navigateur web et accédez à http://localhost:8080 pour visualiser votre fichier index.html.
